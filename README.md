# 🎉 College Event Management System

A responsive web application for managing college events where students can view upcoming events, register for events, check schedules, browse galleries, view FAQs, and contact event coordinators.

---

## 🚀 Features

### 🏠 Home Page
- Responsive Navigation Bar
- Hero Section
- Event Highlights
- Statistics Cards
- Featured Events
- Call-to-Action Section

### 📅 Events Page
- Technical Quiz
- Hackathon
- Poster Presentation
- Coding Contest
- Project Expo
- Cultural Fest

Each event card contains:
- Event Image
- Event Name
- Event Date
- Event Venue
- Register Button

### 📝 Registration Page
- Student Name
- Roll Number
- Branch
- Email
- Mobile Number
- Event Selection
- Registration Success Message

### 📊 Event Schedule
- Responsive Bootstrap Table
- Table
- Table-striped
- Table-hover

### 🖼️ Gallery Page
- Event Photos
- Bootstrap Cards
- Responsive Images
- Grid Layout

### ❓ FAQ Page
- Bootstrap Accordion
- Common Questions & Answers

### 📞 Contact Page
- Contact Form
- Coordinator Details
- Social Media Links
